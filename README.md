# PORTFOLIO WEBSITE

> Built With

- ReactJS
- Tailwind CSS

## Run Locally

### 1. Clone this repopitory

```
$ git clone https://github.com/pvq2k2/portfolio-react.git 
$ cd portfolio-react
```

### 2. Run the app

```
$ npm run server
```

Create new terminal

```
$ npm i
$ npm run dev
```

The app will run in the development mode.
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
